This example indicates that the style sheet will use the ISO-8859-15 character encoding:
@charset "ISO-8859-15";
This at-rule declares the font family called Example Font, which is used in the statement that follows:
@font-face {
  font-family: "Example Font";
  src: url("http://www.example.com/fonts/example");
}
h1 {
  font-family: "Example Font", sans-serif;
}
Here are examples of the most common usage forms:
@import url("/css/main.css");
@import "local.css";
This rule set will be applied only when the document is printed or viewed in print preview mode:
@media print {
  body {
    padding: 1in;
    border: 0.5pt solid #666;
  }
}
Here are some examples of media queries in action:
@import url(/style.css) all and (color);

@media all and (min-width:800px) {
  ⋮ one or more rule sets…
}
The first example below declares a default namespace; the second declares that the prefix svg can be used to refer to elements from the SVG namespace, "http://www.w3.org/2000/svg":
@namespace "http://www.w3.org/1999/xhtml";
@namespace svg "http://www.w3.org/2000/svg";
This example sets default page margins:
@page {
  margin: 1in 1.5in;
}
